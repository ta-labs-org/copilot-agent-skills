# copilot-agent-skills

## ブロック崩しゲーム

シンプルなブロック崩しゲームです。パドルでボールを跳ね返し、すべてのブロックを破壊してレベルをクリアしましょう。

### 動作手順

1. リポジトリをクローンします。
2. ローカルサーバーを起動します（例: `python -m http.server 8000`）。
3. ブラウザで `http://localhost:8000` にアクセスします。
4. スペースキーでゲームを開始/ポーズします。
5. 左右矢印キーでパドルを移動します。

### ゲームルール

- パドルでボールを跳ね返します。パドルの位置によってボールの角度が変わります。
- ブロックにボールを当てて破壊します。ブロックの辺によって反射方向が変わります。
- すべてのブロックを破壊すると次のレベルへ。
- ボールが下に落ちるとライフが減ります。
- ライフが0になるとゲームオーバー。

### テストケース

- パドル移動: 左右キーでパドルが移動するか確認。
- ボール反射: パドルと壁で反射するか確認。
- ボール角度変更: パドル中央寄りで垂直に跳ね返るか確認（dx範囲-3～3）。
- ブロック破壊: ボールがブロックに当たって消えるか確認。
- ブロック辺判定: ブロックの上下左右で適切に反射するか確認。
- 角当たり: ブロックの角に当たると両軸反転するか確認。
- ボール侵入補正: ブロック内に侵入したら位置補正するか確認。
- スコア更新: ブロック破壊でスコアが増えるか確認。
- レベル遷移: すべてのブロック破壊でレベルアップするか確認。
- ゲームオーバー: ライフ0で停止しUIメッセージ表示するか確認。
- ボールリセット: 速度を保持してリセットするか確認。
- リサイズ: ウィンドウリサイズでCanvas調整、ブロック状態保持するか確認。
- エラーハンドリング: ゲームループでエラーが発生したら停止しメッセージ表示するか確認。

### 技術仕様

- HTML5 Canvasを使用。
- JavaScript ES6クラス。
- TDD（テスト駆動開発）で実装。
